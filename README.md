# フローチャートアプリ概要

## 1. アプリ概要

FlowChartApp は、思考整理や仕様作成前のラフ設計に特化した
軽量フローチャート作成ツール です。

ノードの追加・接続・移動が直感的で、初めて使う人でも迷わない操作性を目指して開発しました。

## 2. 特徴（Feature）

-ノード（処理ブロック）をワンクリックで追加

-黒ポチ（ノード端の接続点）からのみ線が生成され、誤操作を防ぐ

-ノード移動時にも線がリアルタイム追従

-補助線（ガイドライン）がリアルタイムで表示され接続操作が直感的

-JSON によるレイアウト保存・読み込み

-複数規格を扱えるコンボボックスによるノードテンプレ切替

-WinForms ベースで拡張性が高い構造

## 3. 技術的アプローチ（経験アピール）
### ●（1）複数規格のコンボボックス管理

-UI 上で複数種類のノード（処理、条件、開始/終了など）を選択可能にし、
ノードの見た目や情報構造が規格ごとに変わる仕組みを実装。

-コンボボックスで規格変更

-選択中規格のテンプレを用いてノード生成

-データ構造を規格ごとに動的切り替え
→ 汎用的なフローチャート構築が可能

### ●（2）JSON によるデータ管理

-ノードとラインの情報を JSON 化し、復元可能にした。

-ノードID

-ノード位置（X/Y）

-文言

-接続先

-ノード種類（規格）

これによりアプリを閉じても
レイアウトが完全に再現される よう設計。

### ●（3）“黒ポチ”からのみ線を出す接続設計

ノードの四隅または中心に “黒ポチ（接続点）” を用意し、
黒ポチ以外の位置からは線を引けない仕様 を実装。

これにより：

-誤操作が減る

-どこが接続点か視覚的に分かる

-接続線の始点 / 終点が明確になる

UI/UX の改善に大きく寄与したポイント。

### ●（4）補助線のリアルタイム追従（接続ガイド）

接続操作中、マウスカーソルに合わせて
補助線（仮ライン） を描画。

・マウスを動かすたび Invalidate() → OnPaint()  
・始点（黒ポチ）と現在位置の座標から仮線を描画  


接続先を探しやすくなり操作性が大きく向上。

### ●（5）ノード移動時のリアルタイム追従

ノードをドラッグすると、それに接続されたすべての線が
リアルタイムで追従 するように実装。

ノードの移動イベントで位置を更新

再描画処理で接続線の始点/終点を再計算
→ 直感的でストレスのない操作感 に成功。

### ●（6）画面 3 分割構成（必要なら説明追加可能）

左：ノードリスト
中央：キャンバス
右：進捗・プロパティ表示

D&D や進捗表示をリアルタイム同期させる構成も可能な設計。
