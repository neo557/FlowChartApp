# FlowChart Designer（フローチャート作成アプリ）

## 概要
頭の中の思考や企画をノード（ブロック）とリンクで可視化するツールです。各ノードに関連ファイル（Word/Excel/PDF/画像/動画）を添付でき、企画の構造や進捗を一目で把握できます。

## 主な機能
- ノード作成・編集・接続
- ファイル添付（Office / PDF / 画像 / 動画）
- 保存・読み込み（シリアライズ：JSON 等）
- ノードの移動・一括選択
- データ内容をHTMLファイルとして出力

## 改善点（現状）
- ブロックの削除が未実装
- テキストの可読性（フォントサイズ等）に改善余地あり

## 技術的ポイント
- ノード／リンクのデータ構造設計
- UI の動的生成とイベント管理
- 外部ファイルの参照管理

## 今後の予定
- ブロック削除・復元機能の追加
- 文字の可読性向上（拡大縮小対応）
- ノードの階層化（グループ化）

## 使い方
- アプリを立ち上げ、ノードを配置して線で接続してください。
- 各ノードにファイル参照を追加できます（内部保存または参照）
